Task 4 - Rock Paper Scissors Game

Description:
A Python-based Rock Paper Scissors game where the user competes against the computer. The computer makes a random choice, and the game determines the winner based on standard Rock-Paper-Scissors rules.

Features:

* User chooses Rock, Paper, or Scissors
* Random computer choice
* Winner determination
* Score tracking
* Play multiple rounds
* User-friendly interface

Technologies Used:
Python

How to Run:
1. Clone the repository.
2. Navigate to the Task4 folder.
3. Run:
python rock_paper_scissors.py
