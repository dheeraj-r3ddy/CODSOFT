Task 5 - Contact Book

Description:
A Python-based Contact Book application that allows users to store and manage contact information. The application supports adding, viewing, searching, updating, and deleting contacts through a simple command-line interface.

Features:
* Add new contacts
* View all contacts
* Search contacts by name or phone number
* Update contact information
* Delete contacts
* Simple and user-friendly interface

Technologies Used:
Python

How to Run:
1. Clone the repository.
2. Navigate to the Task5 folder.
3. Run:
python contact_book.py
